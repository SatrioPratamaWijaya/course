Hari 2: String, List, dan Tuple

1. String dan Operasi pada String

String adalah tipe data yang digunakan untuk merepresentasikan teks atau urutan karakter. Berikut adalah penjelasan tentang string dan beberapa operasi yang dapat dilakukan pada string:

Penjelasan:

  • Operasi Penggabungan (+): Operasi penggabungan dalam string menggunakan operator +. Ketika operator + diterapkan pada dua string, karakter-karakter dari kedua string akan digabungkan secara berurutan untuk membentuk satu string baru.

  • Operasi pengulangan (`*`) digunakan untuk mengulang string sebanyak jumlah yang ditentukan.
  • Indexing digunakan untuk mengakses karakter individual dalam string berdasarkan posisi indeksnya. Indeks dimulai dari 0.
  • Slicing digunakan untuk mengambil potongan substring dari string berdasarkan rentang indeks tertentu.
  • Metode `upper()` digunakan untuk mengubah semua karakter dalam string menjadi huruf kapital.
  • Metode `lower()` digunakan untuk mengubah semua karakter dalam string menjadi huruf kecil.
  • Metode `split()` digunakan untuk memisahkan string menjadi beberapa bagian berdasarkan delimiter (defaultnya adalah spasi) dan mengembalikan list kata-kata.
  • Metode `join()` digunakan untuk menggabungkan elemen-elemen dalam list menjadi satu string menggunakan pemisah tertentu.

2. List

List adalah tipe data yang digunakan untuk menyimpan kumpulan elemen yang dapat diubah. Berikut adalah penjelasan tentang list dan beberapa operasi yang dapat dilakukan pada list:

Penjelasan:

  • Pada contoh di atas, kita membuat beberapa list dengan berbagai jenis elemen.
  • Elemen dalam list dapat diakses menggunakan indexing, dimulai dari 0.
  • Operasi `len()` digunakan untuk menghitung jumlah elemen dalam list.
  • Operasi `append()` digunakan untuk menambahkan elemen baru ke akhir list.
  • Operasi `extend()` digunakan untuk menambahkan elemen dari list lain ke akhir list.
  • Operasi `insert()` digunakan untuk menyisipkan elemen baru ke posisi tertentu dalam list.
  • Operasi `remove()` digunakan untuk menghapus elemen pertama dengan nilai tertentu dari list.
  • Operasi `pop()` digunakan untuk menghapus dan mengembalikan elemen terakhir dari list.

3. Tuple

Tuple adalah tipe data yang digunakan untuk menyimpan kumpulan elemen yang bers ifat tidak dapat diubah (immutable). Berikut adalah penjelasan tentang tuple:

Penjelasan:

  • Pada contoh di atas, kita membuat beberapa tuple dengan menggunakan tanda kurung () untuk mengelompokkan elemen-elemennya.
  • Elemen dalam tuple dapat diakses menggunakan indexing dan slicing, mirip dengan list.
  • Tuple bersarang adalah tuple yang berisi tuple lain sebagai elemen-elemennya.
  • Perbedaan utama antara tuple dan list adalah tuple bersifat tidak dapat diubah (immutable), sedangkan list bersifat dapat diubah (mutable).
