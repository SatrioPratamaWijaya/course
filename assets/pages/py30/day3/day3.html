Hari 3: Set, Dictionary, dan Fungsi Dasar

1. Pengenalan tipe data set

Set adalah tipe data yang digunakan untuk menyimpan kumpulan elemen yang unik (tidak ada duplikat) dan tidak diurutkan (unordered). Dalam set, urutan elemen tidak dijamin, sehingga tidak dapat mengakses elemen menggunakan indeks. Set menggunakan konsep matematika himpunan dan dapat digunakan untuk melakukan operasi himpunan seperti union, intersection, dan difference.

2. Mengapa set berguna dan apa kelebihannya

Berikut adalah beberapa kegunaan dan kelebihan yang dapat dilakukan pada set:

  • Elemen dalam set unik, artinya tidak ada elemen yang sama/duplikat dalam satu set.
  • Penggunaan set berguna saat kita perlu menyimpan koleksi elemen yang hanya membutuhkan keunikan, seperti daftar kata unik dalam sebuah teks.
  • Kelebihan set adalah kemampuannya untuk melakukan operasi himpunan secara efisien, seperti penggabungan (union), irisan (intersection), perbedaan (difference), dan lainnya.

3. Operasi dasar pada set

Berikut adalah beberapa operasi dasar yang dapat dilakukan pada set:

  1. Union (Gabungan)

    Union (gabungan) adalah operasi yang menghasilkan set baru yang berisi elemen-elemen dari kedua set yang digabungkan tanpa duplikat.

    contoh:

    Penjelasan

    Pada contoh di atas, elemen-elemen dari set1 dan set2 yang digabungkan adalah {1, 2, 3} dan {3, 4, 5}. Karena elemen 3 muncul di kedua set, hasil Union akan menghilangkan duplikat dan menghasilkan set baru {1, 2, 3, 4, 5}.

  2. Intersection (Irisan)

    Intersection (irisan) adalah operasi yang menghasilkan set baru yang berisi elemen-elemen yang ada di kedua set.

    contoh:

    Penjelasan

    Pada contoh di atas, irisan antara set1 dan set2 adalah elemen 3, karena itu adalah satu-satunya elemen yang ada di kedua set.

  3. Difference (Perbedaan)

    Difference (perbedaan) adalah operasi yang menghasilkan set baru yang berisi elemen-elemen yang ada di set pertama tetapi tidak ada di set kedua.

    contoh:

    Penjelasan

    Pada contoh di atas, terdapat dua set yaitu set1 dan set2 yang masing-masing berisi elemen-elemen unik. Operasi set1.difference(set2) digunakan untuk menghitung perbedaan antara set1 dan set2. Output yang dihasilkan adalah {1, 2}, yang merupakan set baru yang berisi elemen-elemen yang ada di set1 tetapi tidak ada di set2.

  4. Metode lain pada set

    Set juga memiliki metode dan operasi lain seperti add, remove, discard, clear, dan lainnya yang dapat digunakan untuk memanipulasi set.
