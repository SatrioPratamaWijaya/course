Hari 3: Set, Dictionary, dan Fungsi Dasar

1. Set

  • Pengenalan tipe data Set

    Set adalah tipe data yang digunakan untuk menyimpan kumpulan elemen yang unik (tidak ada duplikat) dan tidak diurutkan (unordered). Set merupakan implementasi dari struktur data himpunan matematika dalam Python.

    contoh:
    Penjelasan:

    1. Pada contoh di atas, kita membuat beberapa set dengan menggunakan tanda kurung kurawal `{}`.

    2. Set `buah` berisi tiga elemen unik: "apel", "pisang", dan "mangga".

    3. Set `bilangan` berisi lima bilangan bulat unik: 1, 2, 3, 4, dan 5.

    4. Set `campuran` berisi empat elemen dengan tipe data yang berbeda: bilangan bulat, string, boolean, dan float.

  • Mengapa Set Berguna dan Apa Kelebihannya

    1. Set memiliki sifat unik, sehingga tidak ada duplikat dalam set. Ini sangat berguna saat kita ingin menyimpan elemen-elemen unik dari suatu kumpulan data.

      contoh:
      penjelasan

      • Pada contoh di atas, kita memiliki sebuah daftar daftar_buah yang berisi beberapa nama buah, termasuk beberapa duplikat.

      • Kemudian, kita menggunakan konstruktor set() untuk membuat set baru, buah_unik, dengan mengonversi daftar menjadi set.

      • Hasilnya adalah set buah_unik yang hanya berisi elemen-elemen unik dari daftar nama buah, yaitu 'mangga', 'jeruk', 'pisang', 'apel', dan 'anggur'.

      • Duplikat seperti 'apel' dan 'mangga' dihapus karena set hanya menyimpan elemen unik.

    2. Set juga mendukung operasi matematika seperti gabungan (union), irisan (intersection), dan perbedaan (difference). Ini memudahkan dalam melakukan operasi himpunan.
    3. Kelebihan lain dari set adalah kemampuannya untuk melakukan operasi dalam waktu yang konstan (O(1)), tanpa tergantung pada ukuran set.
