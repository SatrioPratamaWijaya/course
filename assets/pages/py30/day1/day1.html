1. Pengenalan Python

Python adalah bahasa pemrograman tingkat tinggi yang populer dan mudah dipelajari. Berikut adalah beberapa poin penting tentang Python:

  • Python adalah bahasa pemrograman interpretatif, artinya kode Python dieksekusi secara langsung oleh interpreter tanpa memerlukan proses kompilasi.
  • Python memiliki sintaksis yang sederhana dan mudah dibaca, sehingga mudah dipahami oleh pemula.
  • Python memiliki dukungan yang luas untuk pengembangan berbagai jenis aplikasi, seperti pengembangan web, analisis data, kecerdasan buatan, dan banyak lagi.

5. Pengendalian Aliran

Pengendalian aliran dalam Python digunakan untuk mengatur jalannya program berdasarkan kondisi atau melakukan pengulangan.

  • Pernyataan Kondisional if-else : Pernyataan kondisional if-else digunakan untuk melakukan percabangan berdasarkan kondisi tertentu. Jika kondisi dalam pernyataan if benar, maka blok kode dalam pernyataan if akan dieksekusi. Jika kondisi dalam pernyataan if salah, maka blok kode dalam pernyataan else akan dieksekusi.
